# Proyecto 2 Parcial - Sistema de Inventario MVC

Sistema web desarrollado en PHP utilizando el patrón MVC para la gestión de productos.

## Tecnologías utilizadas
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript

## Requisitos
- XAMPP o servidor Apache
- PHP 8+
- MySQL

## Instalación

1. Clonar el repositorio:
   git clone https://github.com/GuillermoQuimi23/Proyecto-2Parcial.git

2. Copiar el proyecto en la carpeta htdocs.

3. Crear una base de datos en MySQL.

4. Importar el archivo database.sql ubicado en la carpeta /database.

5. Configurar conexión en config/Database.php.

6. Ejecutar en el navegador:
   http://localhost/inventario-php2/public/index.php
